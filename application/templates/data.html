About the Data

The visualizations created for this project were built usind data from the following sources:

  • Fires from Space: Australia from kaggle by Carlos Paradis.

    Includes information for Australia fires from the 2019 - 2020 fire season as recorded by NASA satellites.

  • List of 2019-2020 bushfires by state (Scraped data using beautiful soup).
  • List of Major Bushfires in Australia (Scraped data using Beautiful Soup).
  • 2019 - 2020 Australia bushfire season (Scraped data using Beautiful Soup).
  • Geojson data for Australia by tonywr71.
  • Australia Climate Data from the Australia Government Bureau of Meteorology.

About the API

The data used for this project was loaded into a MongoDB database and is available through a Python Flask-powered API. For more information, see the Australia Fire API documentation.

The following table shows an example of the data the the API returns. This specific example was used to build the map for this project to help visualize the location and intensity of Australia fires.

acq_date acq_time bright_t31 brightness daynight frp instrument latitude longitude satellite
