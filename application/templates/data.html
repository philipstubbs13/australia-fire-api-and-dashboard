About the Data

The visualizations created for this project were built using data from the following sources:

  • Fires from Space: Australia from kaggle by Carlos Paradis
  • Scraped data using Beautiful Soup from List of 2019-2020 bushfires by state
  • Scraped data using Beautiful Soup from List of Major Bushfires in Australia
  • Scraped data using Beautiful Soup from 2019 - 2020 Australia bushfire season
  • Geojson data for Australia by tonywr71
  • Australia Climate Data from the Australia Government Bureau of Meteorology

About the API

The data used for this project was extracted, transformed, and loaded into a MongoDB database on MongoDB Atlas. This data is available for use through a Python Flask-powered RESTful API. For more information about accessing and using this api, see the Australia Fire API documentation page.
