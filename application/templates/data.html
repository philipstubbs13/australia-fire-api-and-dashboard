About the Data

The visualizations created for this project were built using Open Data from the following sources:

Source
Fires from Space: Australia from kaggle by Carlos Paradis.

Includes information in CSV format for Australian fires from the 2019 - 2020 bushfire season as recorded by NASA satellites.

A list of 2019-2020 bushfires by state from Wikipedia (Scraped data using Beautiful Soup).
List of Major Bushfires in Australia from Wikipedia (Scraped data using Beautiful Soup).
2019 - 2020 Australian bushfire season from Wikipedia (Scraped data using Beautiful Soup).
GeoJSON-formatted Australian state and township border data by tonywr71.
Australia Climate Data from the Australia Government Bureau of Meteorology.

About the API

The data used for this project was loaded into a MongoDB database. It is available to you through a Python Flask-powered API. For more information, see the Australian Fire API documentation.

The following table shows an example of the kinds of data that the API returns. This specific example was used to build the map for this project, which visualizes the locations and intensity of bushfires in the 2019-2020 season.

acq_date acq_time brightness daynight frp instrument latitude longitude satellite bright_t31
