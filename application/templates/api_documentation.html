Australia Fire API

The data used for this project was extracted, transformed, and loaded into a MongoDB database. This data is available for use through a Python Flask-powered API. For more information on the original data sources used, see Data.

Resource URL

http://australia-fire-api-dashboard.herokuapp.com/api/v1.0/

Response Format

JSON

Avaiable Endpoints

Route Description
/fires_modis Returns a list of fires from the 2019 - 2020 Australia fire season that were recorded by the MODIS (Moderate Resolution Imaging Spectroradiometer) satellite sensor.
/fires_viirs Returns a list of fires from the 2019 - 2020 Australia fire season that were recorded by the VIIRS (Visible Infrared Imaging Radiometer Suite) satellite sensor.
/historical_fires Returns a list of past Australia fire seasons.

Example Request

Here is an example of a request to get data about the Australia fires captured by NASA satellites (MODIS). This example uses the d3.json method to retrieve data.

const url = "https://australia-fire-api-dashboard.herokuapp.com/api/v1.0/fires_modis" d3.json(url).then(data => { console.log(data); });
