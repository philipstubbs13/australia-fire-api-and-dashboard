Australian Fire API

The data used for this project was extracted, transformed, and loaded into a MongoDB database. This data is available for use through a Python Flask-powered API. For more information on the original data sources used, see Data.

Resource URL

http://australia-fire-api-dashboard.herokuapp.com/api/v1.0/

Response Format

JSON

Avaiable Endpoints

Route Description
/fires_modis Returns a list of fires from the 2019 - 2020 Australia fire season that were recorded by the MODIS (Moderate Resolution Imaging Spectroradiometer) satellite sensor.
/fires_viirs Returns a list of fires from the 2019 - 2020 Australia fire season that were recorded by the VIIRS (Visible Infrared Imaging Radiometer Suite) satellite sensor.
/fires_modis_geojson Returns a GeoJSON-formatted list of fires from the 2019 - 2020 Australia fire season that were recorded by the MODIS satellite sensor.
/fires_viirs_geojson Returns a GeoJSON-formatted list of fires from the 2019 - 2020 Australia fire season that were recorded by the VIIRS satellite sensor.
/fires_historical Returns a list of damage from past Australia fire seasons and major events.
/fires_bystate Returns aggregated data on each state's damage sustained during the 2019-2020 bushfire season.
/bushfire_season_2019 Returns detailed information about each fire recorded during the 2019-2020 bushfire season.
/fires_time_series Returns daily totals for the number of observed fires over time during the 2019-20 fire season, as captured by NASA satellites. Powers the "Number of Fires" time series chart.
/aus_temp_rainfall Returns average annual temperature and average annual rainfall data for all of Australia, from 1956-2019.

Parameters

Name Required Description Example
limit optional The number of fires to return from the endpoint. Default is 1500.

Can be used with the following endpoints:

  • /fires_modis
  • /fires_viirs
  • /fires_modis_geojson
  • /fires_viirs_geojson

https://australia-fire-api-dashboard.herokuapp.com/api/v1.0/fires_modis?limit=100
start_date optional Returns a list of fires that were captured on or after this date. Specify the date in the following format: YYYY-MM-DD.

Can be used with the following endpoints:

  • /fires_modis
  • /fires_viirs
  • /fires_modis_geojson
  • /fires_viirs_geojson

https://australia-fire-api-dashboard.herokuapp.com/api/v1.0/fires_modis?start_date=2019-12-01
end_date optional Returns a list of fires that were captured on or before this date. Specify the date in the following format: YYYY-MM-DD.

Can be used with the following endpoints:

  • /fires_modis
  • /fires_viirs
  • /fires_modis_geojson
  • /fires_viirs_geojson

https://australia-fire-api-dashboard.herokuapp.com/api/v1.0/fires_modis?end_date=2019-12-01

Example Request

Here is an example of a request to retrieve data about for all fires captured by NASA MODIS satellites, using the d3.json method.

const url = "https://australia-fire-api-dashboard.herokuapp.com/api/v1.0/fires_modis" d3.json(url).then(data => { console.log(data); });
